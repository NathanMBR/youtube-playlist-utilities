# v1.0.1 Patch Notes

## New Features

### Video Substitution
You can now replace an unavailable video in your playlist with a working one, instead of just removing it. When viewing your unavailable videos, each entry now shows a **Substitute** button alongside the existing **Remove** button. Clicking it opens a dialog where you paste any YouTube video URL to use as the replacement. The new video will be inserted at the exact same position in the playlist as the original.

### Browser Support
For users that downloaded the code adn are running it locally, the app can now run directly in a web browser, in addition to the desktop application.

## Improvements

- The video list is now preserved on-screen while a playlist refresh is in progress. Previously the list would disappear and be replaced by a full loading screen; now a loading overlay is shown on top of the existing content, making the experience less disruptive after a removal or substitution.

## Deprecations

- `macOS` builds are no longer provided starting from this version.
